```

2. GitHub → POST /deploy webhook
3. 触发 post-deploy-webhook.sh:
   - git pull
   - 质量检查 (npm run web:gate)
   - rsync 分发到 USA + France VPS
   - 健康检查 (curl /health)
   - 失败则回滚

耗时: 3小时
效果: 部署自动化 100%, 延迟从 5分钟 → 30秒

📍 组件:
- /Users/yumei/scripts/post-deploy-webhook.sh
- /Users/yumei/scripts/webhook-server.js
- GitHub仓库设置中的Webhook
```
