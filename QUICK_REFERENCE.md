## 🛠️ 技术栈

- **Frontend**: Node.js + Vanilla JavaScript (无框架)
- **实时更新**: 滑块input事件 → state更新 → 批量重渲染
- **数据源**: 实时Brent油价 + 用户手动参数
- **持久化**: localStorage (自动保存用户输入)
